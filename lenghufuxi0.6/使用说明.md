# 冷湖知识复习系统使用说明

## 系统概述

冷湖知识复习系统是一个基于Web的知识复习和测试平台，具有以下功能：

- 用户登录和注册
- 章节管理（添加、编辑、删除课程章节）
- 知识点管理（添加、编辑、删除知识点）
- 课程选择和答题
- 排行榜功能
- 主机管理（用户权限管理）
- 网络访问地址查看

## 主机操作步骤

### 1. 启动系统

1. **打开主机电脑**
2. **进入系统目录**：`d:\tools\Trae CN\project\p1\lenghufuxi`
3. **启动服务器**：
   - 双击 `start_server.bat` 文件自动启动前端和后端服务器
   - 或手动运行以下命令：
     ```bash
     # 启动后端API服务器（端口9000）
     python server.py --port 9000
     
     # 启动前端HTTP服务器（端口8888）
     python -m http.server 8888
     ```

### 2. 查看网络访问地址

1. **在主机浏览器中**访问 `http://localhost:8888/quiz-system.html`
2. **登录系统**（使用默认账号或注册新账号）
3. **点击"查看网络访问地址"按钮**
4. **复制显示的网络访问地址**（如：`http://192.168.1.100:8888/quiz-system.html`）
5. **将此地址分享给其他终端用户**

### 3. 管理系统

- **管理章节**：点击"管理章节"按钮，可添加、编辑、删除课程章节
- **管理题库**：点击"管理题库"按钮，可添加、编辑、删除知识点
- **主机管理**：点击"主机管理"按钮，可管理用户权限
- **查看排行榜**：点击"查看排行榜"按钮，可查看用户积分排名

## 其他终端操作步骤

### 1. 访问系统

1. **打开浏览器**（支持Chrome、Firefox、Edge等）
2. **输入网络访问地址**（从主机获取）
3. **按下回车键**访问系统

### 2. 登录系统

1. **在登录界面**输入用户名和密码
2. **点击"登录"按钮**
3. **如果没有账号**，点击"点击注册"进行注册

### 3. 使用系统

- **选择章节**：点击"选择章节"按钮，选择要答题的课程
- **开始答题**：选择课程后点击"开始答题"按钮
- **查看排行榜**：点击"查看排行榜"按钮，查看积分排名
- **退出系统**：点击"注销"按钮退出登录

## 常见问题及解决方案

### 1. 无法访问系统

- **检查网络连接**：确保主机和终端在同一局域网内
- **检查服务器状态**：确保主机上的前端和后端服务器都在运行
- **检查防火墙设置**：确保防火墙没有阻止端口8888和9000

### 2. 登录失败

- **检查用户名和密码**：确保输入正确的用户名和密码
- **检查后端服务器**：确保后端API服务器（端口9000）在运行
- **注册新账号**：如果忘记密码，可重新注册新账号

### 3. 功能按钮无响应

- **刷新页面**：按F5刷新页面
- **检查浏览器控制台**：按F12打开开发者工具，查看控制台是否有错误信息
- **重新登录**：注销后重新登录系统

### 4. 课程显示为空

- **检查章节管理**：主机用户需要先在"管理章节"中添加课程
- **检查权限设置**：主机用户需要在"主机管理"中为用户分配课程权限

## 系统功能说明

### 1. 章节管理

- **添加课程**：创建新的一级课程章节
- **添加子课程**：为现有课程添加二级子课程
- **编辑课程**：修改课程名称和父章节
- **删除课程**：删除不需要的课程

### 2. 知识点管理

- **添加知识点**：创建新的知识点，包含标题、内容、分类、所属章节等信息
- **编辑知识点**：修改现有知识点的信息
- **删除知识点**：删除不需要的知识点

### 3. 课程选择和答题

- **选择课程**：从一级课程中选择要答题的课程
- **开始答题**：选择二级子课程后开始答题
- **答题过程**：系统会自动计时，每道题有30秒答题时间
- **查看结果**：答题完成后显示得分、错题解析和排行榜

### 4. 排行榜功能

- **积分规则**：每答对一题得1分
- **排名显示**：按积分从高到低显示用户排名
- **清空排行榜**：主机用户可清空排行榜数据

### 5. 主机管理

- **用户列表**：查看系统中的所有用户
- **课程权限管理**：为每个用户分配可访问的课程
- **权限保存**：保存用户的课程权限设置

## 技术参数

- **前端服务器**：Python HTTP Server（端口8888）
- **后端服务器**：Flask API Server（端口9000）
- **数据存储**：SQLite数据库（quiz_database.db）
- **网络要求**：局域网内所有终端需在同一网段

## 注意事项

1. **主机电脑**需要保持开机状态，服务器需要持续运行
2. **网络访问地址**可能会因网络环境变化而改变，需要重新获取
3. **数据备份**：定期备份 `quiz_database.db` 文件以防数据丢失
4. **安全建议**：不要在公共网络中使用，仅在局域网内使用
